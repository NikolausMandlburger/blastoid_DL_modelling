Description of files:

dataselection_pTE.Rmd
Rmd notebook for selecting the genomic regions for training pTE accessibility models.
Notebook executed with: R 4.2.0 and Bioconductor 3.15

dataselection_pTE_vs_mTE.Rmd
Rmd notebook for selecting the genomic regions which are differentially open in pTE as compared to mTE.
This set of selected sequences is used for motif discovery analysis. 
Notebook executed with: R 4.2.0 and Bioconductor 3.15
